# 🚀 メイメー セットアップガイド（超初心者向け）

このガイドに従えば、**プログラミング知識ゼロ**でもアプリを公開できます！

---

## 📋 必要なもの（全部無料）

1. ✅ GitHubアカウント
2. ✅ Vercelアカウント
3. ✅ Gemini APIキー
4. ✅ 30分の時間

---

## 🎯 STEP 1: GitHubアカウント作成（5分）

### 1-1. GitHubにアクセス
https://github.com を開く

### 1-2. サインアップ
1. 「Sign up」をクリック
2. メールアドレスを入力
3. パスワードを設定（8文字以上）
4. ユーザー名を決める（例：`yamada-taro`）
5. メール認証を完了

---

## 📦 STEP 2: リポジトリを作成（5分）

### 2-1. 新規リポジトリ作成
1. GitHubにログイン
2. 右上の「+」→「New repository」をクリック
3. 以下を入力：
   - **Repository name**: `meimay-app`
   - **Description**: `メイメー - 赤ちゃんの命名アプリ`
   - **Public** を選択（無料）
   - **Add a README file** にチェック
4. 「Create repository」をクリック

### 2-2. ファイルをアップロード
1. 作成したリポジトリページで「Add file」→「Upload files」
2. ダウンロードした`baby-name-builder`フォルダ内の**全ファイル**をドラッグ＆ドロップ
3. 下の「Commit changes」をクリック

---

## 🔑 STEP 3: Gemini APIキー取得（3分）

### 3-1. Google AI Studioにアクセス
https://makersuite.google.com/app/apikey を開く

### 3-2. APIキーを作成
1. Googleアカウントでログイン
2. 「Create API key」をクリック
3. 「Create API key in new project」を選択
4. キーが表示される（例：`AIzaSyBKQQj...`）
5. **このキーをコピー**（重要！後で使います）

---

## 🌐 STEP 4: Vercelでデプロイ（10分）

### 4-1. Vercelにアクセス
https://vercel.com を開く

### 4-2. GitHubでログイン
1. 「Start Deploying」をクリック
2. 「Continue with GitHub」を選択
3. GitHubアカウントで認証

### 4-3. プロジェクトをインポート
1. 「Add New...」→「Project」をクリック
2. `meimay-app`を探して「Import」をクリック
3. **Configure Project画面**で何も変更せず「Deploy」をクリック
4. デプロイ完了を待つ（1〜2分）

### 4-4. 環境変数を設定
1. デプロイ完了後、「Continue to Dashboard」をクリック
2. 上部メニューの「Settings」をクリック
3. 左メニューの「Environment Variables」をクリック
4. 以下を追加：
   - **Key**: `GEMINI_API_KEY`
   - **Value**: 先ほどコピーしたAPIキー（`AIzaSyBKQQj...`）
   - **Environment**: All（3つ全部にチェック）
5. 「Save」をクリック

### 4-5. 再デプロイ
1. 上部メニューの「Deployments」をクリック
2. 最新のデプロイの右側にある「...」→「Redeploy」をクリック
3. 「Redeploy」を確認
4. 完了を待つ（1〜2分）

---

## 🎉 STEP 5: アプリを使う！

### 5-1. URLを確認
1. Vercelダッシュボードで「Visit」をクリック
2. URLが表示される（例：`https://meimay-app.vercel.app`）
3. このURLを開く → **アプリが動いてます！** 🎊

### 5-2. スマホでアプリ化
#### iPhoneの場合：
1. Safariで上記URLを開く
2. 共有ボタン（□↑）→「ホーム画面に追加」
3. 「メイメー」というアイコンが追加される！

#### Androidの場合：
1. Chromeで上記URLを開く
2. メニュー（⋮）→「ホーム画面に追加」
3. アイコンが追加される！

## 📱 使い方

1. **性別を選択**（男の子/女の子/指定なし）

5. **スワイプ開始！**

   - 👆 上：お気に入り

7. **ビルド画面**で組み合わせ
8. **姓名判断**を確認
9. **AI由来生成**をクリック
10. **保存**してパートナーと共有！

## 🔧 トラブルシューティング

### Q1: 「Loading...」から進まない
**A:** ブラウザのコンソールを確認（F12キー → Console）  
→ エラーが出ていたら内容をコピーして質問してください

### Q2: AI由来生成がエラーになる
**A:** 以下を確認：
1. Gemini APIキーが正しく設定されているか
2. APIキーの無料枠を使い切っていないか（月15リクエスト）
3. Vercelで再デプロイしたか

### Q3: スワイプが動かない
**A:** スマホで試してください（PCではマウスドラッグ）

### Q4: 漢字が表示されない
**A:** ブラウザをリロード（F5キー）してください
